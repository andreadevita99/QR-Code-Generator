# 📱 QR Code Generator

## **Instantly generate QR Codes**

## ✨ Key features :

 - 📄 Free text;
 - 🌐 URL / web site;
 - 📶 Wi‑Fi network (with WPA/WPA2, WEP or no encryption);
 - ✉️ e-mail (with optional object and body);
 - 📞 Phone;
 - Social profiles:
   - 📸 Instagram;
   - 🔗 LinkedIn;
   - 🐙 GitHub;
 - 🌐 Bilingual interface (Italian/English);
 - 🌓 Light/dark theme with an iOS/iPhone style toggle;
 - 📱 Responsive and mobile‑friendly design;
 - Download the QR code as a PNG file and copy the image to the clipboard:
   -  ⬇️ Export the QR code as a PNG;
   -  📋 Copy the image to the clipboard (native support)

## **Advanced customisation** :
  - QR Code sizes (150px, 250px, 350px);
  - Error correction level (L, M, Q, H) – explained in the information section.

## **Privacy and security** :
  - All data is processed locally in the browser – nothing is sent to external servers.
